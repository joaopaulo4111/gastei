💸 Gastei

✅ v1.0.0

Aplicação CLI em Java para controle de gastos pessoais com cotação do dólar em tempo real.

📋 Problema Real

Muitas pessoas perdem o controle financeiro por não registrar gastos no dia a dia. O Gastei resolve isso de forma simples e rápida.

✅ Funcionalidades

💰 Adicionar gastos com descrição, valor e categoria
📋 Listar todos os gastos registrados
📊 Ver o total gasto
💵 Cotação do dólar em tempo real via AwesomeAPI
💾 Persistência local em arquivo JSON

⚙️ Como executar

git clone https://github.com/joaopaulo4111/gastei.git
cd gastei
mvn compile exec:java "-Dexec.mainClass=com.gastei.Main"

🧪 Rodar testes

mvn test
